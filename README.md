*Note: The project is under development. It is not ready for deployment yet.*

DataHub
=====
### Dependencies
* `thrift (0.9.1)`
* `postgres (9.3)`

### Running DataHub Server
* `cd src/datahub`
* `source setup.sh`
* `python server.py`


### Running DataHub Interactive Shell
* `cd src/datahub`
* `source setup.sh`
* `python tools/cli.py`

### Running DataHub Example Code
* `cd examples/<...>`
* `source setup.sh`
* execute the `<example-binary>`
