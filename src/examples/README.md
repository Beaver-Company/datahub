### Java Sample Code
* `cd java`
* `source setup.sh`
* `java DHClient``

### C++ Sample Code
* `cd cpp`
* `source setup.sh`
* `make`
* `./DHClient`

### Python Sample Code
* `cd python`
* `source setup.sh`
* `python DHClient.py`

### Go Sample Code
* `cd go`
* `sh run.sh`

### Javascript Sample Code
* `cd javascript`
* open `SampleClient.html` in a browser
