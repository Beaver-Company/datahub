DataHub
=======
### Dependencies
* `thrift`
* `postgres`

### Running DataHub Server
* `source setup.sh`
* `python server.py`


### Running DataHub Interactive Shell
* `source setup.sh`
* `python tools/cli.py`

### Running DataHub Example Code
* `cd examples/<...>`
* `source setup.sh`
* execute the `<example-binary>`
