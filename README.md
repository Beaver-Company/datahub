DataHub
=======
### Dependencies
* `thrift`
* `postgres`

### Runnning DataHub Server
* `cd src/datahub`
* `source setup.sh`
* `python server.py`


### Running DataHub Interactive Shell
* `cd src/datahub`
* `source setup.sh`
* `python tools/cli.py`

### Running DataHub Example Code
* `cd src/datahub/examples/<python | java | javascript | ...>`
* `source setup.sh`
* `execute the <example-binary>`
